Neural Networks
Basic Neural Network Overview

Computers are efficient for doing specific calculations and tasks much faster than humans, such as complex math problems. Unfortunately, basic computers also have limitations. A human can easily do some elementary functions that cannot always be done by a computer, such as determining whether a given image is either a car or a boat (Heaton, 2008). By combining the functionalities of a human brain and a computer, we will be able to train a computer to do some of the tasks a human can. This can be done by creating neural networks, which are computer systems modelled on the human brain and nervous system (Smith et al., 2020). Neural networks can be trained to detect specific patterns in data and then give output results on these patterns (Tu, 1996). Thus, by using neural networks, it is also possible to train a computer to detect the difference between a car and a boat and other types of patterns.

Multi-Layer Neural Network

A multi-layer neural network consists of an input layer, an output layer, and one or more hidden layers between the input and the output layer. Information or data transfers from the input layer through the hidden layer(s) towards the output layer. By using hidden layers, the network can extract higher-order patterns that predict and compute more accurate outputs.
